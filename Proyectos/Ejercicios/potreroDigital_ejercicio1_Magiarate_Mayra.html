Etiquetas Principales para Definir la estructura standar de un doc HTML

  1. '<'!DOCTYPE html>' Indica que se trata de un documento HTML5
  2. '<'html lang = "en" dir = "ltr"'>' Indica que aqui comienza el documento Html, el atributo Lang indica lengua o idioma del documento, y el atributo dir indica la direccion de los textos, en este caso de izq a derecha
  3. <'<' head > Indica o Especifica los metadatos del documento que serviran al navegador, y por eso no se muestran en la pagina
  4. '<'title'> Es una etiqueta dentro de la etiqueta head, que dira a nuestro navegador como se llama la pagina o documento que acabamos de crear
  5. '<'body > Es la etiqueta o elemento que Indica que desde aqui irà todo lo visible en la pagina web o documento html.
