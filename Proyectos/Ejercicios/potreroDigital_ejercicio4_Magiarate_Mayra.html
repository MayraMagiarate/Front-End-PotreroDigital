Atributo de Directorio HTML

Uso del "dir" como atributo de Etiqueta

El atributo dir especifica la dirección del texto del contenido del elemento.

Valores

  • ltr: Direccion del texto de izquierda a derecha (como se ve en esta página).
  • rtl: Direccion del texto de derecha a izquierda.
  • auto: deja que el navegador configure la direcion del texto basandose en su contenido.
  • Item 4 para cumplir con los requisitos
  • Item 5 para cumplir con los requisitos

Ejemplos:

  1. ltr: Este texto se lee de izquierda a derecha.
  2. auto: Este texto lo acomoda el Navegador según contenido.
  3. rtl: Este texto se leee de derecha a izquierda.
